Introducción

La Raspberry Pi es una serie de ordenadores de placa SBC (Single Board Computer) de coste y tamaño reducido.
El concepto es el de un ordenador desnudo de todos los accesorios que se pueden eliminar sin que afecte al funcionamiento básico.
Está formada por una placa que soporta varios componentes necesarios en un ordenador común y es capaz de comportarse como tal.
Desde su lanzamiento en 2012 no ha dejado de ganar popularidad, gracias al amplio rango de posibilidades que ofrece este sistema.
Ahora mismo el ordenador en miniatura va por su cuarta versión y esta siendo utilizada por numerosos proyectos tecnológicos en todo el mundo.
En este blog, veremos algunos de los proyectos mas interesantes que podemos hacer nosotros mismos desde casa si disponemos de una de estas placas.

Estación meteorológica con Raspberry PI

Si eres un principiante en el desarrollo de proyectos con Raspberry, quizás este sea uno de los mejores proyectos para ti.
Al construir una estación meteorológica que recopile y analice información atmosférica, no solo aprenderás los fundamentos básicos de Raspberry PI
si no que además incrementarás tus conocimientos para futuros proyectos.
Puedes elegir entre una gran cantidad de sensores físicos y librerías complementarias de Python para controlar sus funcionalidades.

Highlights:
  • Elejir una placa Raspberry con modulo "Wireless" o usar un "dongle" Wi-Fi.
  • Recomendamos el sensor BME280 para el analisis de la presión, temperatura y humedad.
  • Puedes usar una impresora 3D para imprimir la carcasa

Construye un router TOR

Si eres un "techy", es posible que sepas algo sobre la "dark web".
Tor es un explorador que nos permite acceder a esta cara oculta de Internet.
El famoso buscador utiliza cientos, quizás miles de diferentes dispositivos como routers para asegurar que el trafico permanece anónimo.
Construir un router TOR con una vieja Raspberry es uno de los mejores proyectos para los amantes del movimiento "open source".

Highlights:
  • Recomendamos el uso de Raspbian OS para los proyectos de Raspberry que requieran anonimato.
  • Asegurate de que el servicio TOR arranca automaticamente al inicio.
  • Puedes usar una impresora 3D para imprimir la carcasa
